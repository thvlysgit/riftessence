## Guide d'Utilisation

### Où poster les messages

1. **Message Global**: 
   - Serveurs Discord LoL généraux (#annonces, #général)
   - Reddit r/leagueoflegends, r/summonerschool
   - Forums Riot Games
   - Twitter/X avec hashtags #LeagueOfLegends #LoL #TeamLiquid

2. **Messages Spécialisés** (Coaching/LFT/LFD):
   - Canaux thématiques correspondants
   - Serveurs de niche (coaching, teams, solo queue)
   - Communautés Discord régionales (FR/EU)

3. **Message Réassurance**:
   - En commentaire épinglé sous annonce principale
   - FAQ channel
   - Réponse directe aux questions de sécurité

### Timing Recommandé

- **Meilleur moment**: Soir (19h-23h CET) en semaine, après-midi weekend
- **Jours préférés**: Mercredi-Jeudi-Vendredi (pics d'activité LoL)
- **Éviter**: Lundi matin, très tôt/tard dans la nuit

### Personnalisation

Adaptez les messages selon:
- Ton du serveur (casual vs compétitif)
- Règles de self-promotion
- Présence de bots de modération anti-spam
- Ajoutez captures d'écran si autorisé (voir `/assets/og-image-template.html`)

### Suivi

Après publication:
- Monitorer les retours dans les 48h
- Répondre rapidement aux questions
- Noter les serveurs/canaux les plus réceptifs
- Itérer sur le message selon feedback
